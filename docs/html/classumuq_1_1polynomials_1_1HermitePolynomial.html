Detailed Description

template<typename RealType>
class umuq::polynomials::HermitePolynomial< RealType >

Hermite Polynomials.

Template Parameters
RealTypeFloating-point data type

\( H_n(x) \) is the physicist's Hermite polynomial of degree \( n \) and argument \( x. \)

The first few physicists' Hermite polynomials are
n \( ~~~~~H_n(x) \)
0 \( ~~1~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ \)
1 \( ~~2~x~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ \)
2 \( ~~4~x^2~-~~~~2~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ \)
3 \( ~~8~x^3~-~~~~12~x~~~~~~~~~~~~~~~~~~~~~~~~~~ \)
4 \( ~~16~x^4~-~~~48~x^2~+~~~~12~~~~~~~~~~~~~~~~ \)
5 \( ~~32~x^5~-~~~160~x^3~+~~~120~x~~~~~~~~~~~~~ \)
6 \( ~~64~x^6~-~~~480~x^4~+~~~720~x^2~-~~~120~~~ \)
7 \( ~~128~x^7~-~~1344~x^5~+~~3360~x^3~-~~1680~x \)


https://en.wikipedia.org/wiki/Hermite_polynomials

The results of this class is similar to the multivariate monomials with the degree of r in a space of d dimensions.

See also
umuq::polynomials::polynomial

A Hermite monomial in \( 1 \) variable \( x \) is simply any (non-negative integer) series of \( H_n(x) \):

\( H_0(x), H_1(x), H_2(x), H_3(x), \cdots, H_r(x) \)

The highest exponent of \( x \) is termed the degree of the Hermite monomial.

Here, \(\alpha=\) all of the Hermite monomials in a d dimensional space, with total degree r.

For example: 

*       d = 2
*       r = 2
*
*       alpha[ 0],[ 1] = 0, 0 = H_0(x) H_0(y) = 1
*       alpha[ 2],[ 3] = 1, 0 = H_1(x) H_0(y) = 2x
*       alpha[ 4],[ 5] = 0, 1 = H_0(x) H_1(y) = 2y
*       alpha[ 6],[ 7] = 2, 0 = H_2(x) H_0(y) = (4x^2-2)
*       alpha[ 8],[ 9] = 1, 1 = H_1(x) H_1(y) = 4xy
*       alpha[10],[11] = 0, 2 = H_0(x) H_2(y) = (4y^2-2)
*
*       monomialBasis_(d=2,r=2)   = {1,   H_1(x), H_1(y), H_2(x), H_1(x)H_1(y),  H_2(y)}
*                           alpha = {0,0,   1,0,  0,1,      2,0,      1,1,       0,2}
*
*
*       d = 3
*       r = 2
*
*       alpha[ 0],[ 1],[ 2] = 0, 0, 0 = H_0(x) H_0(y) H_0(z) = 1
*       alpha[ 3],[ 4],[ 5] = 1, 0, 0 = H_1(x) H_0(y) H_0(z) = 2x
*       alpha[ 6],[ 7],[ 8] = 0, 1, 0 = H_0(x) H_1(y) H_0(z) = 2y
*       alpha[ 9],[10],[11] = 0, 0, 1 = H_0(x) H_0(y) H_1(z) = 2z
*       alpha[12],[13],[14] = 2, 0, 0 = H_2(x) H_0(y) H_0(z) = (4x^2-2)
*       alpha[15],[16],[17] = 1, 1, 0 = H_1(x) H_1(y) H_0(z) = 4xy
*       alpha[18],[19],[20] = 1, 0, 1 = H_1(x) H_0(y) H_1(z) = 4xz
*       alpha[21],[22],[23] = 0, 2, 0 = H_0(x) H_2(y) H_0(z) = (4y^2-2)
*       alpha[24],[25],[26] = 0, 1, 1 = H_0(x) H_1(y) H_1(z) = 4yz
*       alpha[27],[28],[29] = 0, 0, 2 = H_0(x) H_0(y) H_2(z) = (4z^2-2)
