Detailed Description

template<typename RealType = double>
class umuq::randomdist::multivariateNormalDistribution< RealType >

Multivariate normal distribution.

Template Parameters
RealTypeFloating-point data type

The multivariate normal distribution of a \(n \text{-dimensional}\) random vector \( {\mathbf X} = \left[X_1, \cdots, X_n \right]^T \) can be written in the following notation:
\( {\mathbf X} \sim \mathcal{N} (\mu, \Sigma) \)
with \(n \text{-dimensional}\) mean vector
\( \mu = {\mathbf E}[{\mathbf X}] = [{\mathbf E}[X_1], \cdots, {\mathbf E}[X_n]]^T, \) and \( n \times n \) covariance matrix
\( \Sigma= {\mathbf E}[({\mathbf X}-\mu)({\mathbf X}-\mu)^T] = [\text{Covariance}[X_i, X_j]; 1\le i,j \le n] \)
