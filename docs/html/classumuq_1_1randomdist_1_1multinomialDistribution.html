Detailed Description

template<typename RealType = double>
class umuq::randomdist::multinomialDistribution< RealType >

Generates random numbers according to a multinomial distribution \( M_K\left(N, p\right) \).

Template Parameters
RealTypeFloating-point data type

Let \( X=\left( X_1, \cdots, X_K \right) \) have a multinomial distribution \( M_K\left(N, p\right) \)
The distribution of \( X \) is given by:
\( Pr(X_1=n_1, \cdots, X_K=n_K) = \frac{N!}{\left(n_1! n_2! \cdots n_K! \right)} p_1^{n_1} p_2^{n_2} \cdots p_K^{n_K} \)

where \( n_1, \cdots n_K \) are nonnegative integers satisfying \( sum_{i=1}^{K} {n_i} = N\),
and \(p = \left(p_1, \cdots, p_K\right)\) is a probability distribution.

Random variates are generated using the conditional binomial method.
This scales well with N and does not require a setup step.

Reference:
C.S. David, The computer generation of multinomial random variates,
Comp. Stat. Data Anal. 16 (1993) 205-217
