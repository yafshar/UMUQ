Detailed Description

This class parses string of data to seperate words.

It ignores all white spaces, tabs, \( : \) and \( , \) characters

void umuq::parser::parse ( const char *  inputLine)

Parses the input line into tokens.

Word separators (delimiters) Characters that will be used as word separators when parsing White spaces, tabs, \(:\) and \(,\).

First, it traverses all white spaces, tabs, \( : \) and \( , \) characters until it hits different character which indicates the beginning of an argument. It saves the address to argv[], and increase the line argument number and skips all characters of this argument.
