Detailed Description

template<typename DataType>
class umuq::residual< DataType >

Computes residuals of observation and predicted data based on different Error type.

Template Parameters
DataTypeData type

List of available Error types:

  • AbsoluteError Absolute difference between observed and predicted data
  • ScaledError It is a ratio of absolute difference between observed and predicted data to the absolute value of observed data
  • SquaredError Squared value of the difference between observed and predicted data
