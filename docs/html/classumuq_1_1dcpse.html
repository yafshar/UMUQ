Detailed Description

template<typename RealType, umuq::DistanceTypes DistanceType = umuq::DistanceTypes::L2, class PolynomialType = polynomial<RealType>>
class umuq::dcpse< RealType, DistanceType, PolynomialType >

General class for (DC-PSE) Discretization-Corrected PSE Operators .

Template Parameters
RealTypeFloating-point data type
DistanceTypeDistance type for finding k nearest neighbors. (Default is a specialized class - kNearestNeighbor<RealType> with L2 distance)
See also
umuq::DistanceTypes
umuq::kNearestNeighbor.
Template Parameters
PolynomialTypePolynomial type used in building the vandermonde & vandermonde-like matrix (Default is - polynomial<RealType> with monomials)
See also
umuq::polynomials::PolynomialTypes.

It creates a discretized differential operator and interpolators.

Todo:
Currently the class works only for one term and it should be extended to multi terms

Evaluate a discretized DC-PSE operator from function values of input data and put the results as the query data function values.

This function uses function values of input data and the weights of the operator which have been previously computed to compute the query values and put the results as the query data function values.
At first it checks the computed kernel size to be equal to the number of query points times the size of monomials which has been previously computed for the required degree of the DC-PSE operator.

Computes generalized DC-PSE differential operators on set of input points.

This function uses one set of points as input data to compute the generalized DC-PSE differential operators.
If the degree of the differential operator is zero \( | \beta | = 0 \), suggests one should use the interpolator function not this one.

Parameters
dataPointsInput data points
nDataPointsNumber of data points
betaIn multi-dimensional notation \( \beta=\left(\beta_1, \cdots, \beta_d \right). \)
Notation for partial derivatives:
\( D^\beta = \frac{\partial^{|\beta|}}{\partial x_1^{\beta_1} \partial x_2^{\beta_2}\cdots\partial x_d^{\beta_d}}. \)
orderOrder of accuracy (default is 2nd order accurate)
nENNNumber of extra nearest neighbors to aid in case of singularity of the Vandermonde matrix (default is 2)
ratioThe \( \frac{h}{\epsilon} \) the default vale is one
