Detailed Description

General class for an interface to the DFT-FE code output.

3

An interface to the DFT-FE code. DFT-FE is a C++ code for material modeling from first principles using Kohn-Sham density functional theory. It handles all-electron and pseudopotential calculations in the same framework while accommodating arbitrary boundary conditions.
