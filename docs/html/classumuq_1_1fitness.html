Detailed Description

template<typename DataType>
class umuq::fitness< DataType >

This class evalutes the model fitness.

List of available Fitness type:

  • errorFitSum Sum of the absolute difference between observed and predicted data.
  • errorFitMean Average of the absolute difference between observed and predicted data.
  • errorFitRootMean Squared root of the average of the absolute difference between observed and predicted data.
  • errorFitMax Maximum value of the absolute difference between observed and predicted data.
