void umuq::tmcmc::optimizationParameters::reset ( )

Reset the optimization parameters which is used in the function optimizer.

The default values are:

  • maxIter = 100
  • display = 0 (Off)
  • functionMinimizerType = 2 (SIMPLEXNM2, The Simplex method of Nelder and Mead)
  • tolerance = 1e-6
  • step = 1e-5
