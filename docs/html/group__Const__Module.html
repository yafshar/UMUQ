Detailed Description

This is the costant module of UMUQ providing constant variable definitions. Constant values: \( \pi,~2\pi,~\sqrt{\pi},~\sqrt{2\pi},~ln(\pi),~\text{and}~ln(2\pi)~\) are defined in this module.
