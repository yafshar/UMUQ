Detailed Description

template<unsigned int A, unsigned int N>
struct CTpowseries< A, N >

CTpowseries<A,N> computes \( 1+A+A^2+A^3+A^4+A^5 \cdots + A^(N-1) \) mod \( 2^32 \). We do NOT use the more elegant formula \( \frac{(a^N-1)}{(a-1)} \) (see Knuth 3.2.1), because it's more awkward to compute with implicit mod \( 2^32 \). Based on recursion:

* g(A,n)= (1+A)*g(A*A, n/2);      if n is even
* g(A,n)= 1+A*(1+A)*g(A*A, n/2);  if n is ODD (since n/2 truncates)
