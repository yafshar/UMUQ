Detailed Description

template<typename DataType, class FunctionType>
class umuq::density::densityFunction< DataType, FunctionType >

Density function class.

Template Parameters
DataTypeData type
FunctionTypeFunction type

Density function or a probability density (PDF), is a function, with a value at any given point (or sample point) interpreted as a relative likelihood that the value of the random variable would be equal to that sample.
