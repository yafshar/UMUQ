Detailed Description

template<class kernelType, class polynomialTailType>
class umuq::radialBasisFunctionCap< kernelType, polynomialTailType >

Capped radial basis function interpolant.

Template Parameters
kernelTypeRadial basis function kernel type (Cubic is default)
polynomialTailTypePolynomial tail type (Linear is default)

This is a capped version of the RBF interpolant that is useful in cases where there are large function values. This version replaces all of the function values that are above the median of the function values by the value of the median.
