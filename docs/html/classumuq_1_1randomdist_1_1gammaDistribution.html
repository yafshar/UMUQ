Detailed Description

template<typename RealType = double>
class umuq::randomdist::gammaDistribution< RealType >

Generates random positive values x, distributed according to probability density function \( \frac{1}{\Gamma (\alpha) \beta^\alpha}x^{\alpha-1}e^{\frac{-x}{\beta}}.\) where \( \alpha > 0 \) is known as the shape parameter and \( \beta > 0 \) is known as the scale parameter.

Template Parameters
RealTypeFloating-point data type
Note
  • \( \alpha > 0 \)
  • \( \beta > 0 \)
