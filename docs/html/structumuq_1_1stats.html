Detailed Description

stats is a class which includes some functionality for statistics of the input data

It includes:

  • minelement Finds the smallest element in the array of data
  • maxelement Finds the greatest element in the array of data
  • minelement_index Finds the position of the smallest element in the array of data
  • maxelement_index Finds the position of the greatest element in the array of data
  • sum Computes the sum of the elements in the array of data
  • sumAbs Computes the sum of the absolute value of the elements in the array of data
  • mean Computes the mean of the elements in the array of data
  • median Computes the median of the elements in the array of data
  • medianAbs Computes the median absolute deviation (MAD) of the elements in the array of data
  • stddev Computes the standard deviation of the elements in the array of data
  • coefvar Computes the coefficient of variation (CV)
  • minmaxNormal Scales the numeric data using the MinMax normalization method
  • zscoreNormal Scales the numeric data using the Z-score normalization method
  • robustzscoreNormal Scales the numeric data using the robust Z-score normalization method
  • covariance Compute the covariance
  • correlation Compute the correlation coefficient (The population Pearson correlation coefficient.)
  • unique Eliminates all but the first element from every consecutive sample points, Find the unique n-dimensions sample points in an array of nRows * nCols data

Correlation (scaler value) between iArray and jArray vectors

Compute the correlation between iArray and jArray vectors which must both be of the same length nSize
Correlation coefficient when applied to a population is commonly represented by the Greek letter \( \rho \) and may be referred to as the population correlation coefficient or the population Pearson correlation coefficient. \( \rho (iArray, jArray) = correlation (iArray, jArray) = \frac{covariance (iArray, jArray) }{stddev(iArray) stddev(jArray) } \) It computes the correlation in one pass of the data and makes use of the algorithm described in Welford~[where] it uses a numerically stable recurrence to compute a sum of products:
\( S = \sum_{i=1}^{nSize} {[(idata_i - \overline{iArray}) \times (jdata_i - \overline{jArray}) ]}, \)
with the relation
\( S_n = S_{n-1} + ((n-1)/n) * (idata_n - \overline{iArray}_{n-1}) * (jdata_n - \overline{jArray}_{n-1}). \)

Reference:
B. P. Welford, "Note on a Method for Calculating Corrected Sums of Squares and Products", Technometrics, Vol 4, No 3, 1962.

Correlation (scaler value) between idata and jdata vectors

Compute the correlation between idata and jdata vectors which must both be of the same length nSize
Correlation coefficient when applied to a population is commonly represented by the Greek letter \( \rho \) and may be referred to as the population correlation coefficient or the population Pearson correlation coefficient. \( \rho (idata, jdata) = correlation (idata, jdata) = \frac{covariance (idata, jdata) }{stddev(idata) stddev(jdata) } \) It computes the correlation in one pass of the data and makes use of the algorithm described in Welford~[where] it uses a numerically stable recurrence to compute a sum of products:
\( S = \sum_{i=1}^{nSize} {[(idata_i - \overline{idata}) \times (jdata_i - \overline{jdata}) ]}, \)
with the relation
\( S_n = S_{n-1} + ((n-1)/n) * (idata_n - \overline{idata}_{n-1}) * (jdata_n - \overline{jdata}_{n-1}). \)

Reference:
B. P. Welford, "Note on a Method for Calculating Corrected Sums of Squares and Products", Technometrics, Vol 4, No 3, 1962.
