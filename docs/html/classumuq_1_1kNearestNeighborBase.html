Detailed Description

template<typename DataType, class FlannDistanceType>
class umuq::kNearestNeighborBase< DataType, FlannDistanceType >

Finding K nearest neighbors in high dimensional spaces.

Template Parameters
DataTypeData type
FlannDistanceTypeDistance type from flann library for computing the distances to the nearest neighbors

Distance type for computing the distances to the nearest neighbors include below distance types from flann library:

  • flann::L2 Squared Euclidean distance functor, optimized version.
  • flann::L2_Simple Squared Euclidean distance functor.
  • flann::L1 Manhattan distance functor, optimized version.
  • flann::MinkowskiDistance The Minkowsky \( (L_p) (x,y) = \left( \sum_{i = 1}^{n} | x_i - y_i |^p \right)^{\frac{1}{p}} \) distance between two vectors \( x,~\text{and}~y. \)
  • flann::MaxDistance The Maximum distance.
  • flann::HistIntersectionDistance The histogram intersection distance.
  • flann::HellingerDistance The Hellinger distance, quantify the similarity between two probability distributions.
  • flann::ChiSquareDistance The distance between two histograms.
  • flann::KL_Divergence The Kullback-Leibler divergence.
  • flann::Hamming Hamming distance functor.
  • flann::HammingLUT Hamming distance functor - counts the bit differences between two strings - useful for the Brief descriptor bit count of A exclusive XOR'ed with B.
  • flann::HammingPopcnt Hamming distance functor (pop count between two binary vectors, i.e. xor them and count the number of bits set).

◆ NearestNeighbors() [1/2]

template<typename DataType , class FlannDistanceType >
int * umuq::kNearestNeighborBase< DataType, FlannDistanceType >::NearestNeighbors ( ) const
inline

A pointer to all points nearest neighbors indices.

The function returns a pointer of size(nPoints * (kNeighbors+1)) all points neighbors.
The returned pointer looks like below:

*    0                1      .     kNeighbors
*   ---------------------------------
*  | 0               0_1     .     0_k
*  | 1               1_1     .     1_k
*  | .
*  | nPoints-1        .      .     (nPoints-1)_k
*

Each row has the size of n which is the number of k neighbors + 1
and it has nPoints rows.
The first column is the indices of points themselves in case of input data and query data are the same.

Returns
All points nearest neighbors indices (in row order).
