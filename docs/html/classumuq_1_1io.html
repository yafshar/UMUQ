Detailed Description

This class includes IO functionality.

Todo:
All the functions should use the ioFormat for printing, save and load and we should remove all options from them.

List of available functions:

  • isFileOpened Check if the file is opened
  • isFileExist Check if the file exists
  • openFile Opens the file with the vailable file open flags:
    • app Seeks to the end of stream before each write
    • binary Open in binary mode
    • in Open for reading
    • out Open for writing
    • trunc Discard the contents of the stream when opening
    • ate Seeks to the end of stream immediately after open
  • readLine Get the string from stream file
  • rewindFile Set the position of stream to the beginning of a file
  • closeFile Close the file
  • getFstream Get the stream
  • getLine Get the Line
  • setPrecision Set the stream Precision
  • setWidth Set the width parameter of the stream to exactly n.
  • getWidth Get the width parameter of the input data and stream for the precision
  • saveMatrix Helper function to save the matrix into a file
  • loadMatrix Helper function to load a matrix from a file
  • printMatrix Helper function to print the matrix to the output stream

Opens the file whose name is specified with the parameter filename.

Opens the file whose name is specified in the parameter filename and associates it with a stream that can be identified in future operations by the FILE pointer returned.

Available file open flags:

  • std::fstream::app Seeks to the end of stream before each write
  • std::fstream::binary Open in binary mode
  • std::fstream::in Open for reading
  • std::fstream::out Open for writing

  • std::fstream::ate Seeks to the end of stream immediately after open
