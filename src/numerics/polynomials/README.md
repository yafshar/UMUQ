Polynomials Module
------------

The Module contains all the polynomial classes in %UMUQ

Contents
----------------

It should contain the following files:

````
    polynomials
    |-- chebyshevpolynomial.hpp
    |-- hermitepolynomial.hpp
    |-- legendrepolynomial.hpp
    |-- polynomial.hpp
    |-- polynomialbase.hpp
    `-- README.md
````
